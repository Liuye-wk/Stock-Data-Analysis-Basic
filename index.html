股票数据分析项目

技术栈： Python, Pandas, NumPy, Matplotlib, Jupyter Notebook

项目简介

这是一个用于分析股票数据的基础项目，包含数据获取、清洗、可视化等功能。

如何使用

# 克隆项目
git clone https://github.com/您的用户名/Stock-Data-Analysis-Basic.git

# 安装依赖
pip install pandas numpy matplotlib

# 运行Jupyter notebook
jupyter notebook stock_analysis.ipynb
