# Stock-Data-Analysis-Basic
A beginner-friendly project to analyze stock data using Python.

# 股票数据分析项目

## 项目简介
使用Python进行的完整金融数据分析，包含数据处理、可视化、风险收益分析和投资组合优化。

## 项目文件
- `stock_analysis.ipynb` - 完整的分析代码
- `stock_analysis_demo.png` - 综合分析图表
- `correlation_heatmap.png` - 相关性热力图
- `efficient_frontier.png` - 投资组合有效前沿

## 技术栈
- Python (pandas, numpy, matplotlib, seaborn)
- Jupyter Notebook
- 金融数据分析方法

## 分析内容
1. 股票数据生成与处理
2. 收益率计算与可视化
3. 风险指标分析（夏普比率、波动率）
4. 相关性分析
5. 投资组合优化与有效前沿
